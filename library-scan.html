Getting Started üöÄ

1. Download Library Scan

2. Pin to Bookmarks Bar for Easy Access

3. Connect Your Goodreads Account

Navigate to your Goodreads profile page. The number listed within the page URL is your ID number‚Äîenter it into Library Scan to connect your Goodreads account.

ex) https://www.goodreads.com/user/show/12345678-isaac

Your account must also be public, a setting which can be modified here.

4. Connect Your OverDrive Library

Please enter the entire URL of your OverDrive library. For example, the New York Public Library can be found at https://nypl.overdrive.com.

Unsure? You can search for your local library here.

5. Start Scanning!

Toggle any eBooks and/or audiobooks preferences and Submit to start scanning!

FAQ üôã

Can I connect multiple OverDrive libraries?

Right now, Library Scan only supports one OverDrive library at a time.

Does my Goodreads profile need to be public?

Yes‚Äîyour account must be publicly viewable so Library Scan can see the books you've shelved. You can make this privacy change here by setting "Who Can View My Profile" to "anyone (including search engines)".

If you would like to keep a Goodreads profile private, you can always revert these privacy changes immediately after searching with Library Scan. However, please be aware that automatic daily scans will stop working once a profile becomes private.
